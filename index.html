Web Developer Work

Here is a sample of web applications showcasing my abilities as a developer. They range from JavaScript games to full stack node.js applications built on express

A Star Wars themed Hangman game. Guess a correct letter to start filling in the dashes. Game tracks your wins and losses. Built with JavaScript and Bootstrap.

A full stack application built using node.js and express to help you find your next best friend. Answer a few questions and get matched with the greatest compatible person.

A fun game built with Javascript and jQuery that utilizes randomly generated numbers. Game tracks your wins and losses.

A website to help connect you with a charity in need. Search by category or location to find a good cause to contribute to. Built with OrgHunter and Google Maps APIs.

Are you a psychic? Guess the randomly generated letter the computer is thinking of. Game tracks your wins and losses. Built with Bootstrap and JavaSript.

A Simpsons themed trivia game. Answer questions about the show before the timer runs out. Game scores your results. Built with Bootstrap, JavaScript, and JQuery.

GifTastic uses the GIPHY API to make a dynamic web page that populates with gifs of your choice. Build with AJAX, API, Bootstrap, jQuery, and JavaScript.

New York Times article search uses an AJAX call to the New York Tines API to compile articles based upon your search criteria.

This application is a train scheduler that incorporates Firebase to host arrival and departure times. The app utilizes Moment.js to provide accurate arrival times and minutes until the train arrives.

This application is a node.js word guess command line game using constructor functions. Play and guess what IMDB top 250 movie the computer has randomly selected.

Bamazon is a node.js application that creates an Amazon-like storefront. The app takes in orders from customers and modifies the inventory using MySQL. There are three different views of the storefront, one for each a customer, manager, and supervisor.

LIRI is a language interpretation and recognition interface. It is a command line node app that does several things. It uses the Twitter API, Spotify API, and OMDB API to take in parameters and give back data. This includes searching for movies, a song, or retrieving most recent tweets.
